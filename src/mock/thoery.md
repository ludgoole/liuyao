# 断卦步骤
## 一、实证对轨
### 1.月令(过去)日令(现在)
+ 日月持世——日常
+ 日月生世——受益
+ 日月冲世——坏处
+ 日月克世——受损
+ 日月合世——好处
+ 日月长生——长存
+ 日月帝旺——鼎盛
+ 日月为墓——受困
+ 日月为绝——绝望
### 2.持世之爻
+ 现状——兄弟持世，目前破财
+ 预期——兄弟持世，将会破财
### 3.暗动之爻
+ 旬空暗动——正在发生
+ 伏神暗动——飞神代表现状
+ 暗动生克冲合——同日月
### 4.旬空之爻
+ 尚未发生
+ 不存在
### 5.主卦卦象
+ 主卦卦象——现在
+ 变卦卦象——将来
### 6.心念爻
+ 初看世爻：世爻未动，变爻不一致
> 与变爻相合之爻，如：妻财未土与子孙午火相合，所想之事与子孙有关
+ 次看伏爻：世爻发动，伏爻不一致
> 与伏爻相合之爻，如：官鬼午火与父母未土相合，所想之事与考试有关
+ 再看君爻：世爻未动，变爻一致，君爻未动，与世爻不一致
> 君爻自身，如：官鬼申金，代表担忧
+ 最后君爻伏爻：世爻发动，君爻发动，伏爻不一致
> 伏爻自身，伏爻所墓之爻，如：子孙未土，代表求医问药；未土为用神父母卯木墓，想知道父亲的寿寝正终

## 三、吉凶定性
### 1.无用动爻
+ 动变回头克
+ 动而化退神
+ 动而化月破
+ 动而化日散
+ 动而化绝
### 2.矛盾趋势
#### 2.1 动能轨迹终点
+ 动始变终
> 变爻遭破散，结局衰败
+ 动变回头生克，变爻为中转
> 按动爻回头作用论，破散为假，即假化破，假化散
#### 2.2 重内轻外
+ 日月为外因，如破散
+ 动变为内因，如化进退，化绝，化回头生克
#### 2.3 世爻为用
+ 六亲定性，如化鬼为真，化月建为假；化子孙为真，化月破为假
### 3.动变组合
#### 3.1 趋旺显吉
+ 爻动变回头生
+ 爻动而化进神
+ 爻动化月建合(不被回头克，化退，化墓)
+ 动变三合相生
+ 有用动爻相生
#### 3.2 趋衰显凶
+ 爻动变回头克
+ 爻动而化退神
+ 爻动而化绝地
+ 爻动而化破散
+ 有用动爻相克
+ 动变三合相克
#### 3.3 旺衰摇摆
+ 有用动爻所冲，旺者冲起，衰者冲破
+ 有用动爻所合，旺者合起，衰者合绊
+ 有用动爻所墓，旺者假墓，衰者真墓
#### 3.4 静爻作用
+ 暗动，旺相静爻逢日冲
+ 冲动，旺相静爻动爻冲
### 4.日月组合
#### 4.1 月令旺相(没有生旺)
+ 月建 1
+ 月合 1
+ 月生 0.1
+ 月扶 0.1
#### 4.2 月令平相
+ 水爻气在丑月
+ 木爻气在辰月
+ 火爻气在未月
+ 金爻气在戌月
#### 4.3 月令衰败(没有墓绝，没有合绊)
+ 月破 -1
+ 月克 -0.1
+ 被克 -0.1
+ 被生 -0.1
#### 4.4 日令旺相
+ 日建 1
+ 日合(静爻) 1
+ 日生 0.1
+ 日扶 0.1
+ 日生旺 0.1
#### 4.5 日令平相
+ 克日 0
+ 生日 0
#### 4.6 日令衰败(没有休囚)
+ 日克 -0.1
+ 日绝 -0.1
+ 日墓(真) -0.1
+ 日绊(动爻) -0.1
+ 日破(真) -1
### 5.特殊日月
#### 5.1 旺相式，逢克不衰
+ 月建日建
+ 月建日合(不作日绊论)
+ 月合日建
+ 月合日合
#### 5.2 衰败式，逢生不起
+ 月破日克
+ 月破日墓
+ 月破日绝
#### 5.3 失效
+ 三合局
+ 动能变化，化进，化退，化绝，化破，搭桥
+ 太过卦
+ 时效性
### 6.世用关系
#### 6.1 吉兆
+ 用神旺相，世爻不衰败
#### 6.2 凶兆
+ 世爻变衰
+ 世爻受克
#### 6.3 作用次序
+ 优先用神，其次世爻
> 如寅木发动，冲用神申金，生世爻午火
> 寅木应冲去用神，而非生旺世爻
## 四、应期推断
### 1.静爻
+ 用神(元神|忌神)逢值逢冲
> 用神子水不动
> 旺相逢冲午日应期，衰败逢值子日应期
+ 用神逢同属性
> 用神子水
> 亥日应期
### 2.动爻
+ 逢值逢合
> 酉金发动
> 逢值酉日应期，逢合辰日应期
+ 三合局，逢冲局或补局
> 亥卯未生官鬼巳火
> 逢巳酉丑应期
### 3.变爻
+ 化回头克，逢值逢冲
> 申金化午火
> 逢值申日应期，逢合巳日应期
+ 化合逢冲，化冲逢合
> 午动变未，逢子丑应期
+ 化进神，逢值、逢合或逢化进
> 申金化酉金
> 逢值申日应期，逢合巳日应期，化进酉日应期
+ 化退神，逢两冲或逢化退
> 酉金化申金
> 逢两冲卯寅应期，化退申日应期
+ 化反吟，逢反吟
> 寅木化申金
> 逢反吟寅申应期
+ 化伏吟，逢值逢冲
> 戌土变戌土
> 逢冲辰日应期
### 4.伏爻
+ 逢值、逢合或飞爻逢冲
> 卯木伏藏于午火
> 逢值卯日应期，飞爻逢冲子日应期
### 5.日月
+ 爻月破，逢填实、补破或出破
> 子月午火爻
> 填实午日应期，补破未日应期，出破出了子月
+ 爻月合，后三个月内
> 辰月酉金爻
> 巳午未应期
+ 爻日冲，逢填实、补破或当日
> 寅日申金爻
> 填实申日应期，补破巳日应期
+ 爻旬空，逢填实、冲实或出空
> 午火旬空
> 填实午日应期，冲实子日应期，出空过了午日
+ 爻入墓，逢冲墓、冲爻或出墓
> 午火墓于戌
> 冲墓辰日应期，冲爻子日应期，出墓过了辰子日
+ 用神多现，逢入墓
> 子孙木爻同时出现在日月变动四处
> 逢入墓未日应期
### 6.旺衰
+ 用神吉世爻受克，逢冲克
> 寅月丑土持世
> 冲克世月令，申月应期
+ 用神凶世爻受生，逢冲生
> 寅月午火持世
> 冲生世月令，申月应期
